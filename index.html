Olympic Athlete Weight Distribution (#TidyTuesday 11/05/2019)

Being relatively new to data viz, for my first #TidyTuesday I decided to re-create a graph from r/dataisbeautiful of the height and weight distribution of Olympic athletes.

Original graph here.

Code here.

This was a fun exercise for me; I did a few things for the first time, like using the viridis colour palette for the first time, and fiddling with the grid and axis labels on a graph (shoutout to the original poster, u/takeasecond, for help with the final touches). Additionally, it was satisfying to systematically re-create a close approximation after having worked through the first few chapters of R for Data Science by Garrett Grolemund and Hadley Wickham. I've clearly improved since the beginning of the summer, when I used R for the first time on a bit of a tight schedule to create a few bar graphs, hacking my way to a solution using my background in CS and ~perseverence~.
